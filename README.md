# LSU_DEXA_PREDICTION_MODELS_PUBLIC

Code repository for "Predicting Anthropometric Body Composition Variables Using 3D Optical Imaging and Machine Learning" by Agrahari et al., published in *Frontiers in Bioinformatics*.

This repository contains implementation code for the core models and experiments reported in the paper. Proprietary datasets and auxiliary analyses have been excluded.

**Authors:** Gyaneshwar Agrahari, Kiran Bist, Monika Pandey, Jacob Kapita, Zachary James, Jackson Knox, Steven Heymsfield, Sophia Ramirez, Peter Wolenski, Nadejda Drenska
